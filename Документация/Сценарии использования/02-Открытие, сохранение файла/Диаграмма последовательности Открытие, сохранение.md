### Прецендент: Открыть/сохранить
``` mermaid

sequenceDiagram
    autonumber
    participant Main as Main (UI)
    participant Editor as Editor
    participant Document as Document
    participant ExportManager as ExportManager
    participant PdfAdapter as PdfAdapter
    participant JsonAdapter as JsonAdapter

    %% === ОТКРЫТИЕ ДОКУМЕНТА ===
    Main->>Editor: OpenDocument("example.docx")
    Editor->>Document: constructor(name="example")
    Editor->>Document: Load(path="example.docx")
    Document-->>Editor: структура восстановлена
    Editor-->>Main: Document готов к отображению
    Main->>Main: Отображает содержимое документа

    %% === ИЗМЕНЕНИЕ ===
    Note over Main,Editor: Пользователь редактирует документ

    %% === СОХРАНЕНИЕ В DOCX ===
    Main->>Editor: Export("docx", "output.docx")
    Editor->>ExportManager: Export(Document, "docx", "output.docx")
    ExportManager->>Document: GetText()
    Document-->>ExportManager: текст и структура документа
    ExportManager-->>Editor: сохраняет "output.docx"
    Editor-->>Main: Документ сохранён

    %% === ЭКСПОРТ В PDF ===
    Main->>Editor: Export("pdf", "output.pdf")
    Editor->>ExportManager: Export(Document, "pdf", "output.pdf")
    ExportManager->>Document: GetText()
    Document-->>ExportManager: текст и структура
    ExportManager->>PdfAdapter: ExportFromDocx("output.docx", "output.pdf")
    PdfAdapter-->>ExportManager: PDF успешно создан
    ExportManager-->>Editor: операция завершена
    Editor-->>Main: Сообщение об успешном экспорте в PDF

    %% === ЭКСПОРТ В JSON ===
    Main->>Editor: Export("json", "output.json")
    Editor->>ExportManager: Export(Document, "json", "output.json")
    ExportManager->>Document: GetText()
    Document-->>ExportManager: структура документа
    ExportManager->>JsonAdapter: ExportFromDocx("output.docx", "output.json")
    JsonAdapter-->>ExportManager: JSON успешно создан
    ExportManager-->>Editor: операция завершена
    Editor-->>Main: Сообщение об успешном экспорте в JSON


```
