### Use Case: Open/Save Document

```mermaid
sequenceDiagram
    actor Пользователь
    participant Editor
    participant FileManager

    Пользователь->>Editor: Выбирает "Open"
    Editor->>FileManager: Открывает диалог выбора файла
    Пользователь->>FileManager: Выбирает файл
    FileManager->>Editor: Загружает содержимое
    Пользователь->>Editor: Вносит изменения
    Пользователь->>Editor: "Save" или "Save As"
    Editor->>FileManager: Сохраняет изменения
    FileManager-->>Editor: Подтверждение сохранения
    Editor-->>Пользователь: Сообщение об успешном сохранении

    alt Ошибка
        FileManager-->>Editor: Файл не найден / повреждён
        Editor-->>Пользователь: Сообщает об ошибке
    else Нет прав доступа
        FileManager-->>Editor: Ошибка доступа
        Editor-->>Пользователь: Сообщает об ошибке
    end
