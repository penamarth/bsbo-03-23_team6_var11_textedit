# Словарь терминов
# Глоссарий терминов по теме «Текстовый редактор»

**Тематика:** Проектирование и реализация простой информационной системы — текстового редактора  

---

### Документ (Document)  
- **Определение:** Основной объект редактора, содержащий текстовые данные, структуру документа и метаинформацию (имя, путь, формат).  
- **Связь:** Состоит из абзацев (Paragraph), предложений (Sentence), слов (Word) и букв (Letter).  
- **Пример:** Demo.cs, NewDocument.txt.  

### Абзац (Paragraph)  
- **Определение:** Элемент документа, объединяющий предложения в логическую структуру.  
- **Связь:** Содержит объекты Sentence, управляется Document.  
- **Использование:** Позволяет выделять, удалять и добавлять предложения.

### Предложение (Sentence)  
- **Определение:** Элемент абзаца, содержащий последовательность слов.  
- **Связь:** Состоит из объектов Word.  
- **Использование:** Формирует текст абзаца, участвует в подсветке синтаксиса.

### Слово (Word)  
- **Определение:** Элементарная единица текста, состоящая из букв.  
- **Атрибуты:** isHighlighted — выделение для подсветки.  
- **Связь:** Состоит из объектов Letter.  
- **Использование:** Анализируется SyntaxHighlighter для подсветки.

### Буква (Letter)  
- **Определение:** Минимальная единица текста.  
- **Атрибуты:** value (символ), isHighlighted, hasCursor.  
- **Использование:** Отображение и управление курсором.

### Курсор (Cursor)  
- **Определение:** Указывает текущее положение редактирования в документе.  
- **Методы:** MoveToNext, MoveToPrevious.  
- **Связь:** Управляется Document и CompositeManager.  

### CompositeManager  
- **Определение:** Компонент для обхода и управления структурой документа.  
- **Методы:** Add, Remove, Traverse, HighlightElement, MoveCursor, GetText, ReplaceText, FindText.  
- **Использование:** Универсальный инструмент для работы с иерархией Document → Paragraph → Sentence → Word → Letter.

### Редактор (Editor)  
- **Определение:** Основной компонент пользовательского интерфейса, обеспечивающий работу с документом и всеми подсистемами.  
- **Методы:** NewDocument, InsertText, GetFullText, ReplaceAll, Search, HighlightSyntax, ValidateSyntax, Export, Print, Close.  
- **Связь:** Взаимодействует с Document, CompositeManager, SyntaxHighlighter, PrintManager, ExportManager.

### Пользователь (User)  
- **Определение:** Человек, взаимодействующий с редактором.  
- **Роль:** Создаёт, открывает, редактирует, сохраняет, печатает документы и использует подсветку синтаксиса.  

### Подсветка синтаксиса (Syntax Highlighting)  
- **Определение:** Выделение ключевых слов и конструкций языка программирования цветом.  
- **Компонент:** SyntaxHighlighter.  
- **Связь:** Создаёт HighlightToken и анализирует структуру Document.  
- **Пример:** Ключевые слова C# `using`, `class`, `void`.

### Токен (HighlightToken)  
- **Определение:** Единица текста, выделенная для подсветки синтаксиса.  
- **Атрибуты:** ElementId, Type, Start, Length.  
- **Использование:** Управление визуальным отображением синтаксиса в Editor.

### Ошибка синтаксиса (SyntaxError)  
- **Определение:** Сообщение о несоответствии синтаксису.  
- **Атрибуты:** Position, Message.  
- **Использование:** Показывается пользователю в процессе анализа документа.

### Принтер (Printer) / PrintManager  
- **Определение:** Внешнее устройство или компонент, осуществляющий печать документа.  
- **Методы:** Print, ShowPreview.  
- **Связь:** Использует PrintSettings и создаёт PrintPreview.

### Настройки печати (PrintSettings)  
- **Определение:** Параметры печати документа (количество копий, диапазон страниц, ориентация, двусторонняя печать).  
- **Использование:** Передаются PrintManager для генерации печати и предпросмотра.

### Предпросмотр печати (PrintPreview)  
- **Определение:** Визуальное представление документа перед печатью.  
- **Методы:** Generate(Document, PrintSettings).  

### Экспорт (Export / ExportManager)  
- **Определение:** Сохранение документа в другие форматы (PDF, JSON).  
- **Компоненты:** PdfAdapter, JsonAdapter.  
- **Методы:** Export, SaveToDocx, ExportFromDocx.  

### Формат файла (File Format)  
- **Определение:** Тип документа, определяемый по расширению файла (.txt, .cs, .html).  
- **Использование:** Определяет подсветку синтаксиса и обработку при открытии/сохранении.
